# random-task-generator

## Автор
София Арчакова

## Описание
Приложение генерирует случайные задачи и сохраняет историю.

## Возможности
- Генерация случайной задачи
- Фильтр по типу (Учёба, Спорт, Работа)
- Добавление новых задач
- Сохранение истории в JSON

## Пример использования
1. Выбрать тип задачи
2. Нажать "Сгенерировать задачу"
3. Посмотреть историю

## Тесты
- Проверка пустого ввода
- Проверка фильтрации
